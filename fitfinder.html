The product

Sizefox is a sizing solution built for Shopify, which was launched in 2021. There are two parts to Sizefox, a customer-facing app and a merchant-facing one. I worked on the app for a few months in 2022, specifically doing research for a new merchant-facing feature that was being built.

Problem

Size recommendations were off for some merchants. Data Scientists conceptualized a new feature.

Some merchant clients sent complaints that size recommendations were off, and requested to be able to adjust them. The data scientists in our team designed a feature that would solve this problem, but it has to be configured by users.

Questions

Would merchants understand this feature and be able to use it?

The team's main goal was to find out if merchants would be able to understand the feature and use it. After briefing the team of Project Manager, Client Success, Data Scientists and Engineers, I raised another issue: would they event want this feature, in general, or was this an isolated request?

Solution

We launched a survey and did moderated user tests of the feature.

I planned the entire research and design cycle of the feature, from the first steps up until final testing. Included in the plan were the steps we would take along the way, along with the rationale behind each of them, as well as a timeline estimation that fit the expected launch date.

The feature was already being designed based on client requests, and I chose to start research with both a survey, to gauge wider interest in the feature, and moderated user tests, to learn in depth what merchants thought of the prototyped feature, and how was their experience using it. I also planned a second testing session to test the usability of the feature, in case there were major changes after the first round. This was cancelled due to changes in company priority.

Impact

Complaints reduced by 10%

I confirmed interest in the feature, and uncovered several usability issues. This impacted in design changes and improvements to the interface and user flow that would increase completion rate. The successful implementation of this feature would reduce complaints by 10%, and reduce return rates for our clients.

The product

Fit Finder is the main sizing solution built by Fit Analytics. As described in this article on TechCrunch, “Fit Analytics’ technology lets people enter their own measurements into a tool that uses machine learning to match those dimensions up to the clothes or shoes in question to find the best fit.” As of 2021, Fit Finder served “a number of big retailers, including North Face, Asos, Calvin Klein, Patagonia, Puma, and many more”.

Problem

High Drop-off Rates

One of the screens in the flow had much higher drop off rates than others, and insights from previous research were inconclusive.

Risks

Changes in the flow impacted merchants' revenues

  • Previous studies that suggested drop off rate causes had small samples.
  • Risk in changing the product based on those hypotheses was high.
